Sobre o MovieDB

O MovieDB é um site dedicado a trazer informações detalhadas sobre os filmes mais recentes, populares e clássicos. Ele utiliza a API do The Movie Database (TMDb) para fornecer dados atualizados e precisos sobre títulos de filmes, datas de lançamento, resumos e muito mais.

Nosso objetivo é conectar amantes de cinema com as informações que eles precisam para explorar o mundo do entretenimento.

Este projeto foi desenvolvido utilizando Vue.js via CDN, com Axios para a integração com a API.

Sinta-se à vontade para navegar e explorar o MovieDB!
